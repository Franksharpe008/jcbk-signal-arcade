# JCBK Signal Arcade

Vibrant music flagship with:

- five downloadable one-minute songs
- a cinematic video-led hero
- a playable arcade splash gate
- animated scroll surfaces and hover borders
- a live capture form for new listeners and collaborators

## Routes

- `/`
- `/tracks`
- `/arcade`
- `/visuals`
- `/experience`
- `/downloads`
- `/contact`

## Local commands

```bash
npm install
npm run dev
npm run build
npm run start
```

## Notes

- Audio and visual assets needed for the live experience are already included in `public/media`.
- Lead forwarding is configured through environment variables at deploy time.
